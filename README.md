# dotfiles

My Personal Setup Scripts + Software RC Files

## Current OS Support

- MacOS

### Install MacOS

```
curl -sSLo setup_mac.zsh https://raw.githubusercontent.com/doubletooth/dotfiles/main/os/mac/setup_mac.zsh
chmod +x setup_mac.zsh && ./setup_mac.zsh && rm -f setup_mac.zsh
```
